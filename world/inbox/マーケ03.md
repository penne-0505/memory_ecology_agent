## 流通の基礎理論

- フォード効果
	- 流通部門の生産性の上昇と実質所得の上昇が人口あたり店舗数の増減に与える影響のこと
		- 生産性が上がると店舗密度は減る。
		- 所得の増加は店舗密度を上げる。
		- ただし、生活必需品の店は生産性効果が上回り、店舗密度が低下しやすい
		- 「生産性」は１店舗あたりの売上、とニアリーイコールで捉えてよい
		- 買回品（比較して買うもの。家具、衣類など）や高級品を扱う小売業は所得効果が上回り、店舗密度は上昇しやすい。
		> １店舗あたりの生産性の上昇、とは「八百屋」、「魚屋」が「SM」に統一されるようなケースにおいても生じ得る。これはわかりやすい。

- 発展段階仮説
	- 発展途上国の適応行動は、それぞれの段階で先進諸国との類似の流通システムをもたらす。
		- 小規模店食料品店の店舗数の減少
		- セルフサービスの増加・普及
		- SM,GMS,百貨店の店舗大型化
		- コーポレート/フランチャイズ・チェーンの果たす役割の高まり
		- 大都市と伝統的な小売流通システムが支配的な地域との格差拡大.広がり続ける.
		- これにより、小規模独立小売店のロビイ活動の増加
		- 業務諸過程の改善志向の高まり
		- 競争の激化

#### 小売業態の発展理論

革新的な小売業態の成立と発展を繰り返す
「小売業態革新のプロセス」を明らかにすることを課題とする

 > 流通の基礎理論には体系的な1理論があるというよりも、適応的な方法論的な論説が多くを占める。らしい。そうなんだ。たしかに"..論"とか"..説"はあるけど"..の法則"とかは見ないね
 
 - 小売の輪の仮説
	 - 小売のイノベーションを「1. 低価格,  2. 格上げ, 3. 低価格」として捉える説
	 > 「格上げ」は「トレーディングアップ」とも言うらしい
	 つまりこうなる。
	 1. 革新者が、営業費を削減することで実現された低コストを差別化要因として市場参入
	 2. 営業成功によって、その方式を模倣する小売業者が増加、新しい業態として定着
	 3. 差別化と得意客の維持のために、コストをかけて格上げを行う。営業比率の上昇
	 4. 新業態が高価格の小売業になる。
	 5. 事業機会をみた新たな小売業者が1のように低コスト化する革新を起こして市場参入
	革新を起こして安さで参入 -> 差別化のために対価を格上げ -> 保守化・弱体化のループ。

- 真空地帯論
	- 消費者が小売業者の提供する対価に関心を持ち、小売業者間の競争を前提とする。
	- イノベーションと保守化のサイクルだけが
	- 対価の向上は価格の上昇につながる（とする）
	- 最低の価格と対価水準から徐々に対価を追加するにつれて、顧客は満足するが、ある一点を超えると顧客は満足しなくなる。
	- ![[PXL_20260501_095413328.RAW-01.COVER.jpg]]こういうこと
	- そこそこの価格でも対価が充実していれば顧客は満足する。


- ライフサイクル論
	- PLCを援用して、業態の成立から衰退までを1. 革新期, 2. 発展期, 3. 成熟期, 4. 衰退期
	- ただし衰退して消滅することはあまりない
	- 年月を経るにつれて、サイクルの速度は早まっている(かも)。成熟期までの期間がSMは35,DCは20,のように。
	- 革新期
		- 既存の業態と異なるコンセプトを主張して登場する段階。価格や品揃え、立地、プロモーションなどの優位性の理解によって売上は伸びゆくが、営業の試行錯誤や小規模さによって利益は低水準
	- 発展期
		- 既存市場を切り崩して売上と利益が急成長。新規参入により市場占有率は増加するが競争は激化
	- 成熟期
		- 初期の革新性が薄まり売上上昇率が停滞。組織の大規模化によるコスト増大や不経済さにより利益が低下
	- 衰退期
		- リポジショニングやコンセプトの修正が行われるが、コンセプト自体の競争力が薄まっているため売上と利益が減少。

- 小売アコーディオン仮説
	- 小売業の品揃えの幅に着目。総合的->専門的->総合的を繰り返す周期的な動きをすることをアコーディオンに例える。
	- ただ、実店舗前提なのでオンラインショップなどでは要因の一部が変化している
	- 縮小に向かう要因
		- 総合化に伴う問題を回避しようとする非経済的要因
		- 免許性など法規制や自主規制に伴う制約
		- 総合化による資本能力の不足
		- 総合化に伴う各コストの上昇（人件費、土地代、など）
	万屋 -> 専門店 -> 百貨店 -> ブティック -> スーパー
	というように捉えるとこの仮説が理解できる
	競争の激化に伴って総合化する傾向がある。

まいばすけっとはロビン・フッドの競合になるか？

---

「まいばすけっと」はごく近傍の1kmにも満たないような範囲に対して「夜ごはんの食材」やコンビニ的な少しだけの購買を徒歩で、例えば公共交通機関の最寄りから家にかけての道で寄るような需要や、高齢者の食材購買の需要（移動が少なく済む。コンビニの店舗密度のスーパーであるのではないかと考えます。）をターゲットとしているため、まとめ買いなどのSM的な需要を1~2km、もしくはそれ以上に対して満たすことのできる「ロビン・フッド」はこれの直接的競合とはなり得ないと考えます。ですが、割安にまとめて買うことができるようになることにより、一部顧客層の毎日の小口購買需要が減り、結果的にまいばすけっと側の利益が減る、というやや間接的な影響は生じ得ると考えます。
